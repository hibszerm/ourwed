# OurWed — Project Context

Version: 1.0

---

# Vision

OurWed is a premium web application built specifically for wedding photographers and wedding filmmakers.

The application is designed to become the main operating system for managing a wedding business.

Our goal is not to build another generic CRM.

Our goal is to create software that understands exactly how wedding creators work.

Every screen, every button and every workflow should save time, reduce stress and help users stay organized during the wedding season.

The application should feel calm, elegant and extremely intuitive.

Users should enjoy opening the application every day.

---

# Design Philosophy

The interface must feel:

Elegant

Premium

Minimal

Calm

Modern

Fast

Inspired by products such as:

Apple

Linear

Notion

Raycast

Framer

Stripe Dashboard

The interface should never feel crowded.

White space is part of the design.

Animations should be subtle.

No unnecessary decorations.

No flashy gradients.

No oversized icons.

---

# User Experience

The application should require as few clicks as possible.

Important information should always be visible.

Users should never search for basic information.

The application should answer questions like:

What wedding is next?

What needs attention today?

Who hasn't paid yet?

Which questionnaires are still missing?

What deadlines are coming?

# Visual Style

Premium through simplicity.

Not through decoration.

Large typography.

Rounded corners.

Natural animations.

---

# Performance

Performance is a feature.

---

# Core Principles

Everything starts from a wedding.

Less is better.

Consistency over creativity.
