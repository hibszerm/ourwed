# Sprint 01 — Core Foundation

## Goal

Transform the current prototype into a premium wedding management application focused on photographers and filmmakers.

---

## Tasks

### OW-001

Rebuild Dashboard according to the Product Backlog.

Requirements:

- Greeting

- Upcoming Wedding

- Today's Tasks

- Notifications

- Recent Weddings

No charts.

No statistics.

---

### OW-002

Create Wedding List.

Use elegant cards.

No tables.

Each card contains:

- Couple

- Date

- Days Remaining

- Package

- Workflow Progress

---

### OW-003

Create New Wedding page.

Fields:

- Couple Names

- Wedding Date

- Ceremony Location

- Reception Location

- Package

- Price

- Notes

---

### OW-004

Create adaptive Wedding Details page.

↓

Use premium spacing.

Minimal interface.

Apple + Linear inspiration.

No gradients.

No emoji.

Large typography.

Rounded cards.

Soft shadows.

Animations should be subtle.
