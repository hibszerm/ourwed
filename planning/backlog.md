# Sprint 1 — Core MVP

## OW-001 — Dodawanie nowego ślubu

Priority: Critical

Status: Todo

Description:

User manually creates a new wedding after the couple accepts the offer.

Fields:

- Bride & Groom names

- Wedding date

- Ceremony location (optional)

- Reception location (optional)

- Package

- Price

- Notes

Acceptance Criteria:

- Wedding appears on Dashboard

- Wedding appears in Calendar

- Wedding status = Reservation

- Wedding page is created

- First automatic task is generated
